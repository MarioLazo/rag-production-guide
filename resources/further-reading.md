2. [LangChain RAG Tutorial](https://python.langchain.com/docs/tutorials/rag/) ‚Äî Hands-on introduction
3. [Pinecone Learning Center](https://www.pinecone.io/learn/) ‚Äî Vector search fundamentals

### Intermediate
1. [LlamaIndex Documentation](https://docs.llamaindex.ai/) ‚Äî Data framework deep dive
2. [Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook) ‚Äî Advanced techniques
